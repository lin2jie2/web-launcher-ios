# web-launcher-ios
iOS Swift WebLauncher H5套壳App

* SafariView 打开外部链接
* Promise Style JavaScript 桥接
 * callNativeFunction("function", {data})
 * callNativeClassMethod("class", "method", {data})
 * callNative({callback}, {data})
* Linking 打开链接
 * Linking.canOpenURL(url)
 * Linking.openURL(url)
 * Linking.instantView(url)
